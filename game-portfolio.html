single page dungeon (in-progress)

This single page dungeon was created in my VIST 386 Game Design class. The goal is to create a dungeon that can be placed into any RPG narrative by a Game Master. This dungeon contains interesting characters, tools, and environments. All of this is summarized in one page, front and back. This specific dungeon is set on a spaceship, where the players wake up imprisoned and must find their way out!
